# Entry Debugger 지원 기능 정리

확인일: 2026-05-24  
대상 버전: Entry Debugger `2.4.0`
대상 확장 폴더: `entry-debugger-extension`

이 문서는 현재 확장 프로그램이 지원하는 기능을 제품 소개 웹페이지로 옮기기 쉽도록 정리한 기능 카탈로그입니다.

## 1. 한 줄 소개

Entry Debugger는 엔트리 만들기 화면에서 변수, 리스트, 신호, 장면, 함수 내부 사용 위치, 콘솔 로그, 실험 기능을 더 빠르게 확인하고 조작할 수 있게 해주는 디버깅 보조 확장 프로그램입니다.

## 2. 지원 범위

- 동작 URL: `https://playentry.org/ws/*`
- 권한: `storage`
- 실행 방식: Chrome Manifest V3 content script + page world 주입 스크립트
- 기본 목적: 엔트리 원본 코드를 수정하지 않고, 만들기 화면 안에서 디버깅과 탐색을 빠르게 하는 보조 도구
- 보안 방향: content script는 엔트리 공식 만들기 URL에서만 실행
- 값 수정, 리스트 편집, 스코프 변경, 오브젝트 추가처럼 사용자가 직접 누른 조작은 현재 프로젝트 상태에 반영됨

## 3. 기본 토글 구조

| 기능 | 위치 | 기본값 | 새로고침 필요 | 설명 |
|---|---|---:|---:|---|
| 디버깅 탭 | 확장 팝업 | 켜짐 | 보통 필요 | 엔트리 속성 패널에 `디버깅` 탭 추가 |
| 설정 탭 | 디버깅 탭 내부 | 항상 표시 | 필요 없음 | 기능별 토글 관리 |
| 함수 사용 바로가기 | 설정 탭 | 켜짐 | 보통 필요 | 속성 탭에 함수 내부 사용 위치 표시 |
| 콘솔 디버깅 | 설정 탭 | 켜짐 | 보통 필요 | 말하기 블록에 외치기/로그 옵션 추가 |
| 함수 안에서 개인변수 보기 | 설정 탭 | 켜짐 | 보통 필요 | 함수 편집 중 지역 변수/리스트 선택 가능 |
| 부스트 모드 버튼 | 설정 탭 | 켜짐 | 보통 필요 없음 | 엔진 화면 상단 부스트 토글 표시 |
| 속성 검색으로 찾기 | 설정 탭 | 켜짐 | 보통 필요 | 블록꾸러미와 속성 탭 목록에 검색 UI 추가 |
| 블록 텍스트 복사 | 설정 탭 | 켜짐 | 보통 필요 | 블록 우클릭 메뉴에 텍스트 복사 추가 |
| Alt 단일 블록 드래그 | 설정 탭 | 꺼짐 | 필요 없음 | Alt 키를 누른 채 드래그하면 선택한 블록만 분리 |
| 초고화질 이미지 저장하기 | 설정 탭 | 꺼짐 | 보통 필요 | 블록 이미지 저장 배율을 200~2000%로 조정, 기본 1000% |
| 실험실 탭 | 설정 탭 | 꺼짐 | 보통 필요 | 실험 기능과 기본 변수 디버깅 표시 |

## 4. 디버깅 탭

엔트리 오른쪽 속성 패널에 기존 탭 디자인과 맞춘 `디버깅` 탭을 추가합니다. 탭 내부에서는 현재 프로젝트 상태를 실시간으로 읽어와 변수, 리스트, 신호, 장면을 관리합니다. 상단 오른쪽의 설정 아이콘을 누르면 기능별 설정 탭이 열리고, 설정 화면에서 다시 누르면 직전에 보고 있던 탭으로 돌아갑니다.

### 4.1 변수 탭

- 변수 이름과 현재 값을 실시간 표시
- 값 클릭 후 즉시 수정 가능
- 검색창으로 이름, 값, 스코프 기준 필터링
- 변수 스코프 표시:
  - `일반`
  - `공유`
  - `실시간`
  - `지역: 오브젝트이름`
- 변수 스코프 변경 가능
- 초시계/대답 같은 기본 변수는 일반 변수 목록에서 제외하고 실험실에서 별도 관리

### 4.2 리스트 탭

- 리스트 이름과 항목 개수 표시
- 리스트 펼치기/접기
- 항목 값 수정
- 항목 추가
- 항목 삭제
- 검색창으로 이름과 스코프 기준 필터링
- 리스트 스코프 표시 및 변경:
  - `일반`
  - `공유`
  - `실시간`
  - `지역: 오브젝트이름`

### 4.3 신호 탭

- 프로젝트에 등록된 신호 목록 표시
- 신호 이름 검색
- `신호 보내기` 버튼으로 선택한 신호 즉시 발생

### 4.4 장면 탭

- 프로젝트 장면 목록 표시
- 장면 이름 검색
- `이동` 버튼으로 선택한 장면으로 전환
- 작품 실행 중 장면 이동 시 `장면이 시작되었을 때` 이벤트를 함께 깨우도록 보조

### 4.5 실험실 탭

실험실 탭은 디버깅 탭 안의 `설정` 탭에서 `실험실 탭` 토글이 켜져 있을 때만 보입니다. 기본값은 꺼짐입니다.

`실험실 탭`이 꺼지면 실험실 안의 모든 기능은 즉시 비활성화되고 기본값으로 정규화됩니다. 다시 실험실을 켜면 각 실험 기능은 기본 꺼짐 상태에서 시작합니다. 설정 탭으로 이동한 속성 검색, 블록 텍스트 복사, 초고화질 이미지 저장하기는 실험실 토글과 독립적으로 동작합니다.

실험실에서 제공하는 기본 디버깅 대상:

- `초시계`
- `대답`

지원 기능:

- 현재 값 표시
- 값 직접 수정
- 표시 상태 확인
- 보이기/숨기기 버튼
- 숨기기 동작은 좌표를 `(500, 0)`으로 이동
- 보이기 동작은 좌표를 `(0, 0)`으로 이동

실험실 안의 추가 토글:

| 기능 | 기본값 | 설명 |
|---|---:|---|
| 터보 모드 | 꺼짐 | 엔트리 속도 조절 패널에 `∞` 단계 추가 |
| 다량 이미지 업로더 | 꺼짐 | 실험실 옆에 `업로더` 탭 표시 |
| 함수 보관함 | 꺼짐 | 실험실 옆에 `함수 보관함` 탭 표시, 등록된 함수 템플릿이 없으면 빈 상태 안내 |

### 4.6 업로더 탭

`실험실 탭`과 `다량 이미지 업로더`가 모두 켜져 있을 때만 표시됩니다.

지원 기능:

- 여러 이미지 파일 선택
- 드래그 앤 드롭
- 오브젝트 이름 지정
- 선택한 이미지로 `.eo` 다운로드
- 다운로드한 `.eo`를 엔트리의 `오브젝트 추가하기 > 파일 업로드`로 업로드하도록 안내
- 선택 파일 또는 생성 `.eo`가 10MB를 넘으면 업로드 실패 가능성 경고
- 선택 목록 비우기

지원 입력:

- PNG
- JPG/JPEG
- GIF
- WEBP
- SVG

변환 규칙:

- 비트맵 입력은 엔트리 호환을 위해 PNG로 변환
- SVG는 원본 SVG와 PNG 래스터 이미지, PNG 썸네일을 함께 생성
- BMP는 지원하지 않음
- `.eo` 다운로드 시 파일 확장자는 `.eo`로 저장

### 4.7 함수 보관함 탭

`실험실 탭`과 `함수 보관함`이 모두 켜져 있을 때만 표시됩니다.

`2.4.0` 웹스토어 제출본에는 내장 함수 템플릿이 없습니다. 개발 검증용 `테스트 함수`는 배포 패키지에서 제거했으며, 실제 사용자용 템플릿이 준비되면 별도 검증 후 추가합니다.

지원 기능:

- 등록된 함수 템플릿 목록 표시
- `추가` 버튼으로 현재 Entry 작품의 함수 목록에 템플릿 추가
- 템플릿 추가 시 함수 ID, 블록 ID, 동적 파라미터 타입, 지역변수 ID를 새로 생성
- 함수 편집 중이거나 작품 실행 중일 때는 추가하지 않음

## 5. 함수 사용 바로가기

엔트리 기본 속성 탭은 변수, 리스트, 신호, 함수가 함수 내부에서 사용된 경우 `함수에 조립되어 있어요.` 정도로만 표시합니다. 이 기능은 기존 엔트리 표시 아래에 `함수에서 사용` 섹션을 추가해 실제 사용 위치를 더 자세히 보여줍니다.

지원 대상:

- 변수
- 리스트
- 신호
- 함수

표시 정보:

- 어느 함수에서 사용했는지
- 어떤 블록에서 사용했는지
- 클릭 가능한 바로가기

동작:

- 항목을 클릭하면 해당 함수 편집 화면으로 이동
- 가능한 경우 사용된 블록을 선택 상태로 표시
- 기존 엔트리 속성 패널과 별도 창 없이 통합 표시

아이디어 제공: `kkomaweb.com`

## 6. 콘솔 디버깅

말하기 계열 블록의 모드 드롭다운에 숨겨진 외치기 모드와 콘솔 로그용 옵션을 추가합니다.

대상 블록:

- `말하기`
- `N초간 말하기`

추가 옵션:

| 라벨 | 용도 |
|---|---|
| 외치기 | 엔트리에 이미 구현된 별 모양 말풍선 모드 표시 |
| `[LOG]` | 콘솔 기본 로그 |
| `[INFO]` | 콘솔 정보 로그 스타일 |
| `[WARN]` | 콘솔 경고 로그 스타일 |
| `[ERROR]` | 콘솔 오류 로그 스타일 |
| `[DEBUG]` | 콘솔 디버그 로그 스타일 |

특징:

- 기존 엔트리 `Dialog` 렌더링 기능을 활용
- 콘솔 로그 옵션은 말풍선을 띄우지 않고 콘솔에 출력
- 기존 프로젝트 JSON을 자동 변환하지 않음
- 확장 없이 열면 확장 전용 옵션은 엔트리 기본 UI에서 온전히 표시되지 않을 수 있음

아이디어 제공: `aqu3180.co.kr`

## 7. 함수 안에서 개인변수 보기

엔트리 함수 편집 화면에서는 원래 `이 오브젝트에서 사용` 변수와 리스트가 블록 꾸러미 드롭다운에 잘 보이지 않습니다. 이 기능은 함수 편집 중에도 현재 오브젝트의 개인 변수와 개인 리스트를 선택할 수 있게 합니다.

지원 대상:

- 변수 드롭다운
- 리스트 드롭다운
- 함수 편집 화면 안의 블록 꾸러미

특징:

- 프로젝트 JSON을 직접 수정하지 않음
- 드롭다운 목록 생성만 보조
- 만들어진 블록은 변수/리스트 ID를 참조하므로 확장 없이도 실행 가능
- 확장이 없으면 선택 UI 보조만 사라짐

## 8. 부스트 모드

엔트리의 WebGL 렌더링 옵션을 켜는 기능입니다.

특징:

- 내부적으로 `Entry.options.useWebGL`을 활성화
- Entry 초기화 시점에 반영되어야 하므로 새로고침 필요
- 디버깅 탭 안의 `설정` 탭에서 `부스트 모드 버튼` 토글은 엔진 화면 상단 토글을 보여줄지 결정
- 실제 부스트 모드는 엔진 화면 상단의 `부스트모드` 토글에서 켜고 끔
- 엔진 상단 토글은 실행 페이지 UI에 맞춰 `부스트모드` 텍스트와 초록 스위치로 표시
- 버튼을 클릭하면 Entry 내장 `Entry.toast.warning`으로 `새로고침 해야 반영됩니다.` 안내 표시
- 기본값은 버튼 표시 켜짐, 실제 부스트 꺼짐
- 무거운 그래픽 프로젝트에서 렌더링 성능 개선을 기대할 수 있음

## 9. 터보 모드

실험실 기능입니다. 엔트리 속도 조절 패널에 `∞` 단계를 추가합니다.

특징:

- 속도 조절 패널에 `1`, `15`, `30`, `45`, `60`, `∞` 숫자 표시
- `∞` 선택 시 FPS는 60으로 유지하고 `Entry.isTurbo`만 활성화
- 터보 모드가 켜질 때 속도 버튼이 깜빡여 상태 변화를 표시
- 작품 실행 중에도 다음 tick부터 반영 가능
- 기본값은 꺼짐

## 10. 속성 검색으로 찾기

설정 탭 기능입니다. 블록꾸러미의 변수, 리스트, 신호 관련 블록 드롭다운과 속성 탭 목록에 검색 입력창을 추가합니다.

지원 대상:

- 변수 드롭다운
- 리스트 드롭다운
- 신호 드롭다운
- 속성 탭 목록: `#entryCode > div.entryVariablePanelWorkspace`

하위 설정:

- 블록꾸러미: 기존 `Entry.FieldDropdownDynamic.prototype.renderOptions()` 패치 기반 검색 UI를 사용
- 속성 탭: `.entryVariableListWorkspace` 안의 실제 `.list` 항목을 검색어로 필터링
- 한글 검색: `hangul-search.js`의 공통 matcher로 일반 문자열, 초성, 자모 분해, QWERTY 오입력 검색을 지원

지원 조작:

- 키워드 검색
- 초성 검색
- 자모 분해 검색
- QWERTY 오입력 검색
- 방향키 이동
- Enter 선택
- Escape 닫기

특징:

- 기존 블록 파라미터와 프로젝트 JSON을 변경하지 않음
- 드롭다운 UI만 검색 가능한 형태로 교체
- 속성 탭 검색은 DOM 표시만 필터링하며 변수/리스트/신호/함수 데이터는 변경하지 않음
- `hangul-search.js`는 content script와 page world에 모두 주입되어 디버거 패널, 블록꾸러미 드롭다운, 속성 탭 검색이 같은 매칭 규칙을 사용함
- 기본값은 켜짐

## 11. 블록 텍스트 복사

설정 탭 기능입니다. 엔트리 블록을 우클릭했을 때 기존 메뉴 맨 아래에 `텍스트로 복사하기` 항목을 추가합니다.

동작:

- 현재 우클릭한 블록부터 아래로 이어진 블록 묶음을 텍스트로 변환
- 코드 배경을 우클릭하면 현재 오브젝트의 모든 코드 묶음을 텍스트로 변환
- 전체 코드 복사 결과의 첫 줄은 `# 오브젝트이름 오브젝트의 코드` 형식으로 표시
- C형 블록 안쪽에 조립된 블록은 들여쓰기로 표현
- 화면에 렌더된 블록 텍스트를 우선 사용해 변수/리스트/신호/모양/소리 이름을 표시
- 입력칸과 드롭다운 값은 `(값)` 형태로 감싸 우선순위와 고정 문구를 구분
- 화면 텍스트를 읽을 수 없는 경우에는 JSON 템플릿을 보조로 해석하고, 가능한 값은 id 대신 이름으로 표시
- 클립보드에 복사한 뒤 확장 패널의 미니 토스트로 완료 안내
- 프로젝트 JSON은 수정하지 않음

주의:

- 설정 탭에서 켜고 끌 수 있음
- 기본값은 켜짐
- 복사 결과는 사람이 읽기 위한 텍스트이며, 엔트리에 다시 붙여 넣는 가져오기 포맷은 아님

## 12. Alt 단일 블록 드래그

설정 탭 기능입니다. 연결된 블록 묶음에서 `Alt` 키를 누른 채 드래그하면 선택한 블록만 분리해 이동합니다.

동작:

- `1-2-3-4`에서 `2`를 Alt 드래그하면 `2`만 이동하고 `1-3-4`는 연결 상태를 유지
- `1-2-3-4`에서 첫 블록 `1`을 Alt 드래그하면 `2-3-4`는 원위치에 별도 스택으로 남김
- 실제 드래그 반경을 넘은 뒤에만 분리하므로 `Alt+클릭`만으로는 프로젝트 구조를 변경하지 않음
- 내부적으로 기존 `separateBlock`, `insertBlock`, `moveBlock` 명령을 조합하고 undo가 하나의 사용자 동작처럼 동작하도록 보정
- 프로젝트 JSON은 최종 블록 배치 결과만 반영하며 별도 메타데이터를 추가하지 않음

주의:

- 설정 탭에서 켜고 끌 수 있음
- 기본값은 꺼짐
- 기본 블록 묶음을 대상으로 하며, 블록 꾸러미/읽기 전용/이동 불가 블록은 제외

## 13. 초고화질 이미지 저장하기

설정 탭 기능입니다. 엔트리 기본 `이미지로 저장하기` 동작을 유지하되, 블록 이미지를 생성하는 순간에만 저장 배율을 사용자가 지정한 값으로 보정합니다.

동작:

- 기존 블록 우클릭 메뉴의 `이미지로 저장하기`를 그대로 사용
- 코드 배경 우클릭의 전체 코드 이미지 저장에도 동일하게 적용
- 설정 탭에서 슬라이더와 숫자 입력으로 200%~2000% 사이 배율을 조정
- 기본값은 1000%
- 1000% 이상이면 `다운로드에 오래 걸릴 수 있습니다.` 안내를 검은색으로 표시
- 실제 코드판 확대/축소 상태는 변경하지 않음
- 저장용 SVG 계산에서만 `BlockView.getDataUrl()`의 배율과 bounding box를 설정 배율 기준으로 보정
- 프로젝트 JSON은 수정하지 않음

주의:

- 설정 탭에서 켜고 끌 수 있음
- 기본값은 꺼짐
- 큰 코드 묶음은 이미지 크기와 저장 시간이 크게 증가할 수 있음

## 14. E.DEBUG 변수 콘솔 추적

디버깅 탭이 활성화되어 폴링 중일 때, 프로젝트에 `E.DEBUG`라는 이름의 변수가 있으면 값 변경을 브라우저 콘솔에 자동 출력합니다.

동작:

- `E.DEBUG` 변수를 감지하면 안내 로그 출력
- 값이 바뀔 때마다 `[E.DEBUG]` 접두사로 콘솔 출력
- 변수가 없으면 한 번 안내 메시지 출력
- 브라우저 콘솔 필터에 `[E.DEBUG]`를 입력해 관련 로그만 볼 수 있음

- 기능 변경 안내 문구: `새로고침 해야 반영됩니다.`
